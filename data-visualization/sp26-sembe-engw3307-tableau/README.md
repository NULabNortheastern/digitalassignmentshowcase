<h2>Summary of Module</h2>

This module demonstrates how to use Tableau to format and visualize data. It is a one-day module. The module will go over a tutorial and introduction to Tableau to teach students how to use the tool, clean and manipulate data, and how to create effective visualizations. 

<h2>Learning Goals</h2>

Students will learn how to use Tableau to clean, manipulate, and visualize data. 

<h2>Learning Objectives</h2>

* Learn Tableau terminology and how to navigate the Tableau interface

* Learn how to import, format, clean, and visualize data using Tableau 

* Understand how to implement accessible design choices, such as adding alt text and using colorblind safe pallets

* Understand the strengths and limitations of Tableau public and gain awareness of open-source alternatives
