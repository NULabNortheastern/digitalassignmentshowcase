<h2>Summary of Module</h2>

This module provides introductions to text encoding with TEI/XML, and mapping data with Tableau. 

<h2>Learning Goals</h2>

+ Learn about the use of TEI in digital projects and archives. 
+ Review and practice navigating TEI guidelines and element specifications. 
+ Learn about options for gaining further proficiency with TEI.
+ Understand spatial data structure in the context of Tableau
+ Learn Tableau terminology and understand how Tableau displays data
+ Know how to plot points in Tableau based upon x/y coordinates
+ Understand filtering and visualization options for spatial data
