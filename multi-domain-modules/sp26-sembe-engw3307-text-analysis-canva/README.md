<h2>Summary of Module</h2>

This module provides an introduction to performing computational text analysis and creating infographics. The first half of the module demonstrates how to use the web-based tools Voyant and Lexos for text analysis. The second half demonstrates how to use Canva to create infographics. 

<h2>Learning Goals</h2>

+ Learn fundamental terms and concepts for text analysis 
+ Review how to create and prepare a text corpus for analysis 
+ Learn how to navigate and perform analyses of word patterns in Voyant and Lexos
+ Learn fundamental concepts for creating infographics 
+ Learn how to navigate Canva and create an infographic
+ Understand how to implement accessible design choices, such as adding alt-text, in Canva
